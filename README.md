# Digital-Library
A SQL-based Digital Library Audit System designed to manage book records, student borrowing activities, and track overdue books. The system helps identify students with pending returns and analyzes popular book categories to support better library decisions.

#Features:
Track issued and returned books
Identify overdue books (more than 14 days)
Analyze most borrowed book categories
Remove inactive student records

#Technologies:
SQL (DDL & DML)
Joins and Conditions
Aggregate Functions (COUNT, GROUP BY)
